This package exposes a module that provides all of the tools needed to integrate with the Storj network. You must have Node.js v4.x.x, Python v2.x.x, and Git installed. Complete documentation can be found here.

npm install storj-lib --save

If you want access to the Storj CLI, you must install it separately or use the storj metapackage to install both the core library and command line interface.
